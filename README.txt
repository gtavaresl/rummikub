Rummikub na linguagem C. Jogo de tabuleiro clássico de peças que possui valores e cores diferentes, que tem como função combinar com outras peças de forma lógica, dando a vitória para o jogador que conseguir ficar sem nenhuma carta primeiro, ou caso o baralho de 106 cartas acabe, ganha o jogador que tiver menos pontos na mão.
*************************************************************************
Compilação: 
-GCC:Abra o terminal, ou Prompt de Comando, acesse a pasta onde está o código fonte “main.c”. Para a compilação, execute: “gcc main.c -o rummikub”.
-LINUX:Depois de compilado, execute o comando: “./rummikub”.
-WINDOWS:Depois de compilado, execute o arquivo “rummikub.exe” que foi criado na pasta onde está o programa.
